# 结构预览

MonolithLib 提供了类似 Litematica 的逐层投影预览功能，让你在建造前先预览结构布局。

## 开启预览

```
/ml preview
```

对准 Rebar 控制器使用，会显示投影方块。

## 预览特性

### 性能优化

- **范围限制**：仅渲染玩家周围 7 格范围内的投影方块
- **动态加载**：随着玩家移动，动态加载周围的投影方块
- **零延迟**：即使是巨型结构也不会造成卡顿

### 视觉效果

- **投影方块**：半透明显示，不会遮挡视线
- **颜色编码**：不同材质可能显示不同颜色
- **逐层显示**：类似 Litematica 的层级投影效果

## 分层控制

```
/ml layer up
```
上一层。

```
/ml layer down
```
下一层。

```
/ml layer <数字>
```
跳转到指定层数。

## 操作说明

### 移动视角

在预览模式下，你可以自由移动：
- 使用 WASD 移动
- 使用鼠标调整视角
- 投影方块会跟随你的位置动态更新

### 退出预览

```
/ml preview cancel
```

取消当前预览。

## 对比传统预览

| 特性 | MonolithLib | 传统方式 |
|------|-------------|----------|
| 性能 | 极致优化，巨型结构不卡顿 | 大型结构会严重卡顿 |
| 渲染范围 | 仅渲染周围 7 格 | 渲染整个结构 |
| 内存占用 | 极低 | 较高 |
| 状态显示 | 仅显示材质，忽略状态 | 显示完整方块状态 |

## 最佳实践

1. **先预览后建造**：在放置蓝图前先预览，确认结构布局
2. **分段建造**：对于巨型结构，可以分段预览和建造
3. **多人协作**：多个玩家可以同时预览同一个蓝图，协作建造
4. **分层查看**：使用 `/ml layer` 命令逐层查看复杂结构
