title: 安装 MonolithLib
description: 安装 MonolithLib 并开始使用
---

import { Callout } from 'fumadocs-ui/components/callout';

<Callout type="danger">
  MonolithLib 目前还在研发阶段，不能投入正常使用。
</Callout>

1. 确认你已经安装了 Rebar。
2. 从[这里](https://github.com/mc506lw/MonolithLib/releases/latest)下载最新版本的 MonolithLib。
3. 将 .jar 文件放入插件文件夹并重启服务器。**[不要使用 /reload](https://madelinemiller.dev/blog/problem-with-reload/)**。首次启动会比平时更长，但之后 Rebar/Pylon 的加载速度会快很多。
4. 完成了！
