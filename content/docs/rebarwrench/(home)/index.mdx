RebarWrench 是一个 Minecraft 插件，它为 **Rebar 附属** 提供了一个强大的**扳手工具**。玩家可以使用扳手与 Rebar 方块进行直观交互，而开发者也可以通过简洁的 API 定义可自定义的方块行为。

[🐙 在 GitHub 上找到我们 →](https://github.com/lijinhong11/RebarWrench)
[📦 安装 RebarWrench →](install)

继续往下看。

## ✨ 功能特性

| 状态 | 功能 | 说明 |
| :----- | :------ | :------ |
| ✅ | **扳手工具** | 以 Minecraft 刷子为底材的自定义扳手物品，支持与方块交互而不破坏方块 |
| ✅ | **配置模式** | 右键点击方块触发配置操作（由开发者自定义） |
| ✅ | **旋转模式** | 按住 Shift + 右键点击方块触发旋转操作（由开发者自定义） |
| ✅ | **快速破坏** | 按住 Shift + 左键点击方块可瞬间拆解（需启用该选项） |
| ✅ | **事件回调 API** | 开发者通过 `interactFunction` 回调处理三种操作类型，返回操作结果 |
| ✅ | **多语言支持** | 内置英语、简体中文、繁体中文、葡萄牙语（巴西） |
| ✅ | **Rebar 附属** | 作为标准 Rebar 附属集成，依赖 Rebar 和 Pylon |
| ✅ | **Maven Central 发布** | 可通过 Maven Central 引入依赖 |

## 🎮 工作原理

RebarWrench 使用 Minecraft 的**刷子物品**作为底材。当玩家使用扳手点击一个 Rebar 方块时：

1. 扳手会检查该方块是否已注册了 `Wrenchable`
2. 如果已注册，根据玩家操作确定动作类型：
   - **右键** → 触发 `CONFIGURE`（配置）
   - **Shift + 右键** → 触发 `ROTATE`（旋转）
   - **Shift + 左键** → 触发 `FAST_BREAK`（快速拆解）
3. 调用 `Wrenchable.onWrenchInteract()` 并传入对应的 `WrenchAction`
4. 开发者在回调中根据 `WrenchAction` 类型执行对应逻辑，返回 `WrenchResult`

对于 `FAST_BREAK` 操作，如果返回 `SUCCESS`，方块将被瞬间破坏且不产生掉落物。

## 🏗️ 核心架构

| 组件 | 文件 | 说明 |
| :--- | :--- | :--- |
| **主插件** | [RebarWrench.java](https://github.com/lijinhong11/RebarWrench/blob/main/src/main/java/io/github/lijinhong11/rebarwrench/RebarWrench.java) | 入口点、Rebar 附属注册、可扳手化注册表 |
| **扳手物品** | [TheWrenchItem.java](https://github.com/lijinhong11/RebarWrench/blob/main/src/main/java/io/github/lijinhong11/rebarwrench/wrenches/TheWrenchItem.java) | 扳手交互逻辑、操作分发、快速破坏处理 |
| **刷子监听器** | [BrushListener.java](https://github.com/lijinhong11/RebarWrench/blob/main/src/main/java/io/github/lijinhong11/rebarwrench/listeners/BrushListener.java) | 使用扳手破坏方块时防止掉落物生成 |
| **Wrenchable API** | [Wrenchable.java](https://github.com/lijinhong11/RebarWrench/blob/main/src/main/java/io/github/lijinhong11/rebarwrench/api/Wrenchable.java) | 支持扳手交互的方块接口及 Builder 构造器 |
| **WrenchAction** | [WrenchAction.java](https://github.com/lijinhong11/RebarWrench/blob/main/src/main/java/io/github/lijinhong11/rebarwrench/api/WrenchAction.java) | 操作类型枚举：CONFIGURE / ROTATE / FAST_BREAK |
| **WrenchResult** | [WrenchResult.java](https://github.com/lijinhong11/RebarWrench/blob/main/src/main/java/io/github/lijinhong11/rebarwrench/api/WrenchResult.java) | 操作结果枚举：SUCCESS / FAILED / UNSUPPORTED |
