1. 确保你正在运行 Paper 或其分支。Pylon 不兼容 Spigot。
2. 从[这里](https://github.com/pylonmc/rebar/releases/latest)下载最新版本的 Rebar。
3. 从[这里](https://github.com/pylonmc/pylon/releases/latest)下载最新版本的 Pylon。
4. 将 .jar 文件放入插件文件夹并重启服务器。**[不要使用 /reload](https://madelinemiller.dev/blog/problem-with-reload/)**。首次启动会比平时更长，但之后 Rebar/Pylon 的加载速度会快很多。
5. 查看 Rebar 和 Pylon 的插件文件夹，了解所有可用的配置选项。
6. [安装一些附属组件](list-of-addons.md)。
7. 完成了！

### Rebar 数据存储在哪里？

你可能注意到 Rebar 不使用数据库，并且在其插件文件夹中似乎没有任何存储文件。这是因为 Rebar 像 Minecraft 一样，把所有数据都存储在世界文件自身中！你不需要担心除世界文件之外的其他数据备份：**Rebar 的数据始终与世界数据保持一致。** 这也意味着 Rebar 与其他类似插件相比，更不易发生数据损坏。
