### 已知兼容性问题

以下是与 Rebar 存在已知问题的插件。

| 插件 | 状态 | 详情 |
|------|------|------|
| WorldEdit | <span class="compatibility-status-minor-incompatibilities">轻微不兼容</span> | 正常使用即可，但不要尝试用 WorldEdit 操作 Rebar 方块 |
| Axiom | <span class="compatibility-status-minor-incompatibilities">轻微不兼容</span> | 正常使用即可，但不要尝试用 Axiom 编辑 Rebar 方块 |
| InteractiveChat | <span class="compatibility-status-minor-incompatibilities">轻微不兼容</span> | 正常使用即可，但 Rebar 在 InteractiveChat 中显示的消息有时无法正确翻译 |
| CMI | <span class="compatibility-status-minor-incompatibilities">轻微不兼容</span> | 正常使用即可，但聊天中用 `[item]` 显示的物品无法正确翻译 |
| MCMMO | <span class="compatibility-status-minor-incompatibilities">轻微不兼容</span> | 铁锤无法正常工作，因为 McMMO 认为玩家在右键点击铁块时试图修复物品 |
| CraftEngine | <span class="compatibility-status-moderate-incompatibilities">中度不兼容</span>（有缓解方案） | 会破坏 Rebar 的部分配方。如需修复，可运行 `/rb exposerecipes` 并编辑受影响的配方文件 |
| RoseStacker | <span class="compatibility-status-do-not-use">请勿使用</span> | 与 Rebar 物品存在复制 bug |
