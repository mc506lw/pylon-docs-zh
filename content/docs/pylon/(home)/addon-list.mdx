| Pylon | Pylon 团队 | 添加所有基础的 Pylon 内容。大多数附属组件运行所必需。 | [GitHub](https://github.com/pylonmc/pylon/releases) |
| MonolithLib | mc506lw | 添加巨型多方块机器和精密展示实体的支持。 | [GitHub](https://github.com/mc506lw/monolithlib/releases) |
| Endfield-Industry | mc506lw | 添加明日方舟·终末地风格的集成工业系统。 | [GitHub](https://github.com/mc506lw/endfield-industry/releases) |
| Pylon-Delight | mmmjjkx | 添加农夫乐事风格的食物和物品。 | [GitHub](https://github.com/lijinhong11/pylon-delight/releases) |
| RebarMobs | ybw0014 | 添加实体捕捉系统，可将实体作为物品伙伴。 | [GitHub](https://github.com/GuizhanCraft/RebarMobs/releases) |
| ConstructionWand | balugaq | 添加建筑之杖，方便玩家快速建造和挖掘。 | [GitHub](https://github.com/balugaq/construction-wand/releases) |
| PylonDroid | Intybyte | 添加可编程机器人，使用 PISA 指令执行任务。 | [GitHub](https://github.com/Intybyte/PylonDroid/releases) |
| OccultOperations | JustAHuman | 添加恶魔、仪式、牺牲等黑暗元素。 | 开发中 |
| RebarRank | Idra | 添加基于 Pylon 物品的升级系统。 | 开发中 |
| RebarBlockStats | JustAHuman | 添加服务器方块放置/破坏统计。 | 开发中 |
| Gantry | glomdom | 添加与 Pylon 并行的进度体系，专注于复杂配方链。 | 开发中 |
| Galactipylon | Seggan | 添加火箭、行星探索和天文学研究功能。 | 开发中 |
| Cartographic | JustAHuman | 添加自动映射图册、三维地图、路径点信标等地图功能。 | 开发中 |

> **注意**：Rebar 附属开发目前尚未正式支持。Rebar/Pylon 可能会在后续更新中对 API 进行重大更改，这可能导致你的附属无法正常工作。目前文档也比较少，如果你仍想开发附属，可以查看[教程](/docs/pylon/create-addons/tutorial-0)入门。（需要具备 Java/Kotlin 开发经验）
