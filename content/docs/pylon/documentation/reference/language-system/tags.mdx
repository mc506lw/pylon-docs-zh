Rebar 的语言文件使用 [MiniMessage](https://docs.advntr.dev/minimessage/index.html) 格式。MiniMessage 自带不少标签，Rebar 在此基础上又加了新的。

<Callout type="info">
  标签的详细用法见[教程 3](../../../create-addons/tutorial-3)。
</Callout>

## Rebar 添加的标签列表

| 标签 | 说明 | 示例 | 示例效果 |
| :-- | :---------- | :------ | :----- |
| <span style={{whiteSpace: 'nowrap'}}>`<arrow>`<br />`<arrow:[COLOR]>`</span> | 插入一个右箭头 (→)，可指定颜色（默认：0x666666） | <span style={{whiteSpace: 'nowrap'}}>`<arrow> Watermelon`</span> | ![arrow](img/arrow.webp) |
| <span style={{whiteSpace: 'nowrap'}}>`<guidearrow>`</span> | `<arrow:0x3a293>` 的简写；用于 Rebar 指南 | <span style={{whiteSpace: 'nowrap'}}>`<guidearrow> Watermelon`</span> | ![guidearrow](img/guidearrow.webp) |
| <span style={{whiteSpace: 'nowrap'}}>`<diamond>`<br />`<diamond:[COLOR]>`</span> | 插入一个菱形 (◆)，可指定颜色（默认：0x666666） | <span style={{whiteSpace: 'nowrap'}}>`<diamond> Watermelon`</span> | ![diamond](img/diamond.webp) |
| <span style={{whiteSpace: 'nowrap'}}>`<star>`<br />`<star:[COLOR]>`</span> | 插入一个星形 (★)，可指定颜色（默认：NamedTextColor.BLUE） | <span style={{whiteSpace: 'nowrap'}}>`<star> Watermelon`</span> | ![star](img/star.webp) |
| <span style={{whiteSpace: 'nowrap'}}>`<insn></insn>`</span> | 应用黄色样式 (0xf9d104)，用于指令 | <span style={{whiteSpace: 'nowrap'}}>`<insn>Right click</insn>`</span> | ![insn](img/insn.webp) |
| <span style={{whiteSpace: 'nowrap'}}>`<guideinsn></guideinsn>`</span> | 应用紫色样式 (0xc907f4)，用于指南中的指令 | <span style={{whiteSpace: 'nowrap'}}>`<guideinsn>Right click</guideinsn>`</span> | ![guideinsn](img/guideinsn.webp) |
| <span style={{whiteSpace: 'nowrap'}}>`<attr></attr>`</span> | 应用青色样式 (0xa9d9e8)，用于属性 | <span style={{whiteSpace: 'nowrap'}}>`<attr>Watermelons:</attr> 5`</span> | ![attr](img/attr.webp) |
| <span style={{whiteSpace: 'nowrap'}}>`<unit:[unit]></unit>`<br />`<unit:[prefix]:[unit]></unit>`</span> | 将数字格式化为单位，可带可选公制前缀 | <span style={{whiteSpace: 'nowrap'}}>`<unit:seconds>1000</unit>`</span> | ![unit](img/unit.webp) |
| <span style={{whiteSpace: 'nowrap'}}>`<nbsp></nbsp>`</span> | 将空格替换为不间断空格，防止换行 | <span style={{whiteSpace: 'nowrap'}}>`<nbsp>Watermelon</nbsp>`</span> | ![nbsp](img/nbsp.webp) |
| <span style={{whiteSpace: 'nowrap'}}>`<item:[item_name]>`</span> | 渲染原版或 Rebar 物品的翻译名称 | <span style={{whiteSpace: 'nowrap'}}>`<item:pylon:bandage>`</span> | ![item](img/item.webp) |
| <span style={{whiteSpace: 'nowrap'}}>`<entity:[entity_type]>`</span> | 渲染实体类型的翻译名称 | <span style={{whiteSpace: 'nowrap'}}>`<entity:creeper>`</span> | ![entity](img/entity.webp) |
| <span style={{whiteSpace: 'nowrap'}}>`<effect:[effect_type]>`</span> | 渲染药水效果的翻译名称 | <span style={{whiteSpace: 'nowrap'}}>`<effect:speed>`</span> | ![effect](img/effect.webp) |
| <span style={{whiteSpace: 'nowrap'}}>`<enchant:[enchant_name]>`</span> | 渲染附魔的翻译名称 | <span style={{whiteSpace: 'nowrap'}}>`<enchant:sharpness>`</span> | ![enchant](img/enchant.webp) |
| <span style={{whiteSpace: 'nowrap'}}>`<biome:[biome_name]>`</span> | 渲染生物群系的翻译名称 | <span style={{whiteSpace: 'nowrap'}}>`<biome:plains>`</span> | ![biome](img/biome.webp) |
