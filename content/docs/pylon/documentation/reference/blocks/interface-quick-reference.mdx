- **物流接口**

  允许你的方块与物流系统（包括货运）交互的接口

  如果你想让机器兼容货物插入器/提取器，需要实现 [LogisticRebarBlock](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/LogisticRebarBlock.html)。

  | 接口 | 文档 |
  | :-------: | :------- |
  | CargoRebarBlock | [Javadocs](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/CargoRebarBlock.html) |
  | LogisticRebarBlock | [Javadocs](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/LogisticRebarBlock.html) |

- **流体接口**

  允许你的方块与流体系统交互的接口

  除非必要，应优先使用 [FluidBufferRebarBlock](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/FluidBufferRebarBlock.html) 和 [FluidTankRebarBlock](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/FluidTankRebarBlock.html) 而非 [FluidRebarBlock](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/FluidRebarBlock.html)。[FluidRebarBlock](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/FluidRebarBlock.html) 接口相当抽象，设计上尽可能灵活，需要机器自行实现流体存储逻辑。

  | 接口 | 文档 |
  | :-------: | :------- |
  | FluidRebarBlock | [Javadocs](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/FluidRebarBlock.html) |
  | FluidBufferRebarBlock | [Javadocs](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/FluidBufferRebarBlock.html) |
  | FluidTankRebarBlock | [Javadocs](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/FluidTankRebarBlock.html) |

- **多方块接口**

  将你的方块变为多方块结构的接口

  你创建的大多数多方块可以使用 [SimpleRebarMultiblock](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/SimpleRebarMultiblock.html)。对于更复杂的多方块，可以使用 [RebarMultiblock](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/RebarMultiblock.html)，但该类设计上尽可能灵活，因此非常基础——不包含幻影方块、不考虑旋转等。

  | 接口 | 文档 |
  | :-------: | :------- |
  | RebarMultiblock | [Javadocs](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/RebarMultiblock.html) |
  | SimpleRebarMultiblock | [Javadocs](https://pylonmc.github.io/rebar/docs/javadoc/io/github/pylonmc/rebar/block/interfaces/SimpleRebarMultiblock.html) |
