## 什么是生成器

资源包生成器是名为 `resource_pack_generator.py` 的 Python 脚本

## 运行生成器

目前为了运行生成器你需要自己在本地运行 Python 脚本。
很快我们将有更简单的方法不需要你知道/学习如何运行 Python 脚本，但在此之前这是唯一的方法。

运行脚本后你应该在 `output/` 文件夹下有一个新的压缩好的资源包。
