# 前置要求

在使用 rebar 资源包生成器之前，你应该对 [Minecraft Java 版资源包] 有基本的了解，包括使用和创建它们两个方面。

# 设置

首先访问[本仓库的主页]，然后点击屏幕右上角的绿色"Use this template"按钮。完成选项设置后，你将拥有自己的仓库来开始构建你的资源包了！

例如我们的官方附属组件 [pylon] 就有 [pylon-resource-pack]。

创建好仓库后，将其[克隆][clone]到你的计算机上，你就可以准备[快速入门](getting-started)了！

[Minecraft Java 版资源包]: https://minecraft.wiki/w/Resource_pack#Java_Edition

[克隆]: https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository
